CSS

HTML5

HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para estructurar y presentar contenido en la web. Permite definir elementos como encabezados, párrafos, enlaces, imágenes y tablas, facilitando la creación de páginas y aplicaciones web interactivas.

SQL

SQL (Structured Query Language) es un lenguaje de programación utilizado para gestionar bases de datos relacionales, permitiendo realizar consultas, insertar, actualizar y eliminar datos, definir esquemas y manipular la estructura y contenido de las tablas de manera eficiente.

SPRING

Spring es un framework de desarrollo de aplicaciones empresariales basado en Java. Proporciona un enfoque modular y una amplia gama de características para facilitar el desarrollo, la integración y la implementación de aplicaciones robustas y escalables..
